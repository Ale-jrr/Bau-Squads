# Content Factory Squad
Squad especialista em produção de conteúdo em escala para marketing — planejamento, criação, adaptação e agendamento.

## Comando de Instalação
```bash
npx squads add gutomec/squads-sh-aios/content-factory-squad -y
```

## O que Faz
O **Content Factory Squad** cobre todo o pipeline de produção de conteúdo:
1. **Planejamento Editorial**: Calendários multi-canal e mapeamento de jornada.
2. **Criação de Conteúdo Longo**: Artigos, blog posts, e-books e whitepapers.
3. **Adaptação para Redes Sociais**: Transforma o conteúdo longo em posts para Instagram, Twitter/X, LinkedIn e scripts para TikTok.
4. **Briefs de Imagem**: Direção visual e prompts para geração por IA.
5. **Agendamento Multicanal**: Gestão de fila e cadência de publicação.

## Agentes
- `content-strategist`: Planejamento e orquestração.
- `long-form-writer`: Redação de textos longos e SEO.
- `social-media-adapter`: Adaptação para diversos formatos sociais.
- `image-brief-generator`: Criação de especificações visuais.
- `publishing-scheduler`: Gestão de agendamento e plataformas.

## Workflows Principais
- `*content-pipeline`: Pipeline completo da estratégia à publicação.
- `*social-blast`: Adaptação rápida para todas as redes sociais de uma vez.

## Comandos Principais
- `*plan-calendar`: Planeja o calendário editorial.
- `*write-article`: Escreve um artigo estruturado.
- `*adapt-social`: Gera posts para redes sociais.
- `*batch-briefs`: Gera múltiplos briefs de imagem.
