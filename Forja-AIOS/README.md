# AIOS Forge Squad
Squad supremo de desenvolvimento, otimização e evolução do AIOS — domínio total do framework, Claude Code e auto-atualização contínua.

## Comando de Instalação
```bash
npx squads add gutomec/squads-sh-aios/aios-forge-squad -y
```

## O que é
O **AIOS Forge Squad** é a meta-ferramenta do ecossistema AIOS: um squad que cria, valida, otimiza e moderniza o próprio framework. Ele cobre todo o ciclo de vida de artefatos AIOS — de agentes a squads completos.

## Agentes
- `aios-oracle`: Orquestração e análise profunda.
- `aios-architect`: Design de estruturas.
- `aios-forge`: Criação de agentes, tasks, workflows e skills.
- `aios-sentinel`: Validação e auditoria de frameworks.
- `aios-catalyst`: Otimização e modernização de componentes.
- `aios-nexus`: Integração de configurações e deploy.
- `aios-scout`: Pesquisa de atualizações e auto-atualização de conhecimento.

## Workflows Principais
- `forge_artifact`: Pipeline de criação completo.
- `optimize_framework`: Auditoria e otimização geral.
- `self_update`: Auto-atualização com base em novidades do Claude Code/AIOS.

## Comandos Principais
- `@aios-forge *create-squad`: Cria um squad completo em 7 fases.
- `@aios-sentinel *audit-framework`: Auditoria completa do projeto.
- `@aios-catalyst *modernize-component`: Migra componentes para padrões atuais.
- `@aios-scout *research-updates`: Pesquisa novidades no ecossistema.
