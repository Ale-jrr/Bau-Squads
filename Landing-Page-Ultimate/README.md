# Ultimate Landing Page Squad
Squad fullstack para criação de landing pages de ultra-alta conversão, cobrindo desde o discovery até o QA final.

## Comando de Instalação
```bash
npx squads add gutomec/squads-sh-aios/ultimate-landingpage -y
```

## O que Faz
O **Ultimate Landing Page Squad** utiliza um pipeline de 7 fases para transformar ideias em páginas de alta conversão:
1. **Discovery**: Entendimento do produto e público.
2. **Research**: Análise de mercado e concorrentes.
3. **Copy & Design**: Textos persuasivos e design system atômico.
4. **Images**: Geração de imagens por IA.
5. **Build**: Desenvolvimento Frontend (Next.js) e Backend (FastAPI).
6. **Integrations**: Conexão com WhatsApp e E-mail.
7. **QA**: Revisão de SEO, acessibilidade e performance.

## Agentes Principais
- `lp-strategist` (Strategos): Discovery e escopo.
- `lp-scout` (Scout): Pesquisa e análise.
- `lp-quill` (Quill): Copywriting.
- `lp-prism` (Prism): Design system.
- `lp-lens` (Lens): Geração de imagens.
- `lp-pixel` (Pixel): Desenvolvimento Frontend.
- `lp-forge` (Forge): Desenvolvimento Backend.
- `lp-bridge` (Bridge): Integrações.
- `lp-shield` (Shield): QA e auditoria.

## Comandos Rápidos
- `*run-full-pipeline`: Executa todo o processo do zero.
- `*create-landing-page`: Inicia a criação.
- `*start-discovery`: Inicia apenas a fase de entendimento.
- `*run-qa`: Executa a auditoria final da página.
