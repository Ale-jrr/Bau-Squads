# skeptic-protocol
Implementação do SKEPTIC Protocol (Ceticismo Construtivo) em 5 fases rigorosas para engenharia de software preventiva.

## Comando de Instalação
```bash
npx squads add xgeniusbr/uploads/skeptic-protocol -y
```

## Visão Geral
O **skeptic-protocol** aplica ceticismo construtivo para reduzir falhas antes da implementação:
1. **Previsão de Falhas**: Mapeia modos de falha e riscos antes de codar.
2. **Testes que Falham Primeiro**: Converte riscos em testes negativos (fase Red).
3. **Implementação Guiada por Testes**: Codifica apenas o necessário para passar nos testes.
4. **Apelo Adversarial**: Executa desafio de edge cases e quebra controlada.
5. **Relatório de Veredito**: Consolida evidências e decisão final do ciclo.
